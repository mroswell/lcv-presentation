Key Technology Breakthroughs

   
  1. Replicate the look and feel of a Geographic Information System in JavaScript.
    2. Boundaries, points, thematic colors, zoom, pan, and info boxes.
  2. Populate a map and web page using Google Sheets.
    3. NCLCV is the first.
  3. Consume the "Congress API" on CommitteeMaps.org
    4. 2015: Add state legislative committees, via the "Open States API"
  4. Color a text chart by LCV score.
    Not yet implemented on a live site.
